🚀 Gerador de CSS com IA
Um projeto prático que utiliza a API do Groq (modelo Llama 3.3) para transformar descrições em linguagem natural em código HTML/CSS funcional com visualização em tempo real.

📋 Sobre o Projeto
O usuário descreve uma ideia (ex: "uma bola azul quicando") e a inteligência artificial gera o código necessário. O site então exibe o código-fonte e renderiza o resultado instantaneamente em um preview.

✨ Funcionalidades
Integração com IA: Consumo da API do Groq para geração de código inteligente.

Preview em Tempo Real: Visualização imediata do código gerado dentro de um iframe.

Interface Responsiva: Design moderno e escuro (Dark Mode) desenvolvido com CSS Flexbox.

Feedback de Carregamento: O botão altera seu estado enquanto a IA processa a requisição.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura da página.

CSS3: Estilização com foco em UI/UX moderna e fontes externas (Google Fonts - Inter).

JavaScript (ES6+): Manipulação de DOM, funções assíncronas (async/await) e requisições HTTP (fetch).

Groq API: Processamento de linguagem natural com o modelo llama-3.3-70b-versatile.

🚀 Como Executar
Clone este repositório.

Abra o arquivo index.html no seu navegador.

Digite uma ideia no campo de texto e clique em Gerar Código!.
